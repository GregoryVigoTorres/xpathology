Boolean comparisons

  • in order of precedence
  • or
  • and
  • = !=
  • <=, <, >= >

Operators are left accumulative 3 > 2 > 1 is the same as (3 > 2) > 1. The string-values of items are passed to boolean or number functions for comparison, basically.


A context consists of

  • a node (the context node)
  • a pair of non-zero positive integers (the context position and the context size)
  • a set of variable bindings
  • a function library
  • the set of namespace declarations in scope for the expression


Node Set

  • A node set is returned by a location path expression.
  • the | operator can be used to join node sets

Node Types

  • element
  • attribute
  • text

Numbers

Numbers are floating point numbers including positive negative infinity, positive and negative 0, and Nan.

operators

Object Types
