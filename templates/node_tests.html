Remember axis-name::NODE_TEST[expression]

Every axis has a principal node type.

If the axis can contain elements, its type is element. Otherwise, it can be either

  • attribute, for the attribute axis
  • namespace, for the namespace axis

A node test is true when the name matches the qualified name of a node, e.g. child::p selects child p elements.

* matches all nodes

Node test text() is true for text nodes, e.g. child::text() selects text nodes. {% endblock %}
